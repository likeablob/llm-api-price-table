# [LLM API Price Table](https://likeablob.github.io/llm-api-price-table)

Yet yet another LLM API pricing comparison tool built with Astro 5 and React 19.

## Setup

```bash
mise trust      # Trust the configuration
mise install    # Install Node.js 24.13.0
npm ci          # Install dependencies
```

## Development and Testing

```bash
npm run dev      # Start development server
npm run build    # Build static site
npm run format   # Format code with Prettier
npm run typecheck && npm run lint # Quality checks
npm run test     # Component tests (Storybook + Vitest)
npm run e2e      # E2E tests (Playwright)
```
